The CSS Box Model

Every HTML element is a rectangular box. The CSS Box Model describes how these boxes are rendered on the page, consisting of content, padding, border, and margin.

This demonstration highlights these core components. The `content` is the text and images inside the box. The `padding` is the space between the content and the `border`. The `border` is a line around the padding. Finally, the `margin` is the transparent space outside the border that separates this box from other elements.
